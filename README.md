# Event & Ticket Management Microservice

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).

Microservice responsible for managing events, event areas, tickets, and QR code generation within the Outdoors platform.

## Features
- Event CRUD management
- Event Area management
- Ticket booking (VIP, REGULAR, PREMIUM, STUDENT)
- QR Code generation for tickets
- Service discovery via Eureka
- Inter-service communication via OpenFeign

## Tech Stack
### Backend
- Java 17
- Spring Boot
- Spring Cloud (Eureka, OpenFeign)
- MySQL
- Maven
